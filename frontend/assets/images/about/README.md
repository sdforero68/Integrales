# Imágenes para la sección "Nosotros"

Coloca las siguientes imágenes en esta carpeta:

## Imágenes necesarias:

1. **historia.jpg** - Imagen que represente la historia del emprendimiento (puede ser una foto de cocina, amasando, o una imagen representativa del inicio del negocio)

2. **compromiso.jpg** - Imagen de pan artesanal o productos elaborados (puede ser una foto de pan cortado, productos en una tabla de madera, etc.)

## Tamaños recomendados:

- **historia.jpg**: 800x600px o similar (formato horizontal)
- **compromiso.jpg**: 800x600px o similar (formato horizontal)

## Formato:

- Formato: JPG o PNG
- Calidad: Alta resolución para buena visualización
- Peso: Optimizadas para web (máximo 500KB cada una)

Si no se colocan las imágenes, se mostrará un placeholder automáticamente.
